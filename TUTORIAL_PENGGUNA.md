# Tutorial Pengguna JokiTugas

Panduan ini ditujukan untuk dua jenis pengguna utama:
- `Client`: pengguna yang ingin memposting tugas dan menerima hasil kerja.
- `Joki / Pekerja`: pengguna yang ingin mengambil tugas dan mengerjakannya.

## 1. Cara Memulai

1. Buka halaman utama JokiTugas.
2. Klik tombol `Daftar Akun Gratis` atau `Masuk`.
3. Jika belum punya akun, pilih tab `Daftar`.
4. Isi:
   - nama lengkap
   - email
   - password
   - pilih peran `Client` atau `Joki / Pekerja`
5. Setelah berhasil daftar, Anda akan langsung masuk ke aplikasi.

## 2. Panduan Untuk Client

### A. Isi Saldo Terlebih Dahulu

Sebelum memposting tugas, client harus memiliki saldo yang cukup.

Langkah:
1. Buka menu `Dompet`.
2. Pilih tab `Top Up`.
3. Masukkan nominal top up.
4. Isi metode pembayaran atau catatan transfer.
5. Upload bukti transfer.
6. Tunggu verifikasi admin.

Catatan:
- Jika saldo belum cukup saat posting tugas, sistem akan otomatis mengarahkan Anda ke halaman `Top Up`.
- Total biaya posting tugas adalah `budget + biaya layanan 5%`.

### B. Posting Tugas

Langkah:
1. Buka menu `Posting Tugas`.
2. Isi:
   - judul tugas
   - deskripsi tugas
   - kategori
   - deadline
   - budget
3. Periksa ringkasan biaya.
4. Klik `Posting Tugas`.

Setelah tugas berhasil dibuat:
- tugas akan muncul di `Tugas Saya`
- pekerja dapat melihatnya di `Marketplace`

### C. Memantau Tugas

Buka menu `Tugas Saya`.

Status tugas yang mungkin muncul:
- `Terbuka`: tugas belum diambil pekerja
- `Dikerjakan`: tugas sedang dikerjakan
- `Ditinjau`: hasil sudah dikirim dan menunggu review Anda
- `Selesai`: tugas sudah selesai
- `Dibatalkan`: tugas dibatalkan

### D. Review Hasil Pekerjaan

Jika pekerja sudah mengirim hasil:
1. Buka detail tugas.
2. Baca catatan pengiriman.
3. Download file hasil jika tersedia.
4. Lakukan review.

Kemungkinan tindakan:
- `Terima`: tugas selesai dan pembayaran diproses
- `Minta Revisi`: tugas dikembalikan ke pekerja untuk diperbaiki
- `Tolak`: gunakan hanya jika memang hasil tidak dapat diterima sesuai kebijakan platform

### E. Lihat Riwayat Transaksi

Menu `Transaksi` digunakan untuk melihat:
- top up
- pengeluaran tugas
- transaksi lain yang terkait akun Anda

Jika ada transaksi yang ditolak admin:
- status akan tampil sebagai `Ditolak`
- alasan penolakan juga akan ditampilkan

## 3. Panduan Untuk Joki / Pekerja

### A. Daftar Sebagai Joki

Saat mendaftar:
1. pilih tab `Daftar`
2. pilih peran `Joki / Pekerja`
3. lengkapi data akun

### B. Cari Tugas

Masuk ke menu `Marketplace`.

Di halaman ini Anda bisa:
- melihat semua tugas yang masih terbuka
- mencari tugas berdasarkan kata kunci
- memfilter kategori
- mengurutkan tugas berdasarkan waktu, budget, atau deadline

### C. Ambil Tugas

Langkah:
1. Klik salah satu tugas di `Marketplace`.
2. Baca detail tugas dengan teliti.
3. Jika sesuai, klik `Ambil Tugas`.

Setelah diambil:
- tugas akan masuk ke menu `Pekerjaan Saya`
- status berubah menjadi `Dikerjakan`

### D. Kirim Hasil Pekerjaan

Jika tugas sudah selesai:
1. Buka menu `Pekerjaan Saya`.
2. Pilih tugas yang sedang dikerjakan.
3. Klik `Kirim Hasil Kerja`.
4. Isi catatan pengiriman.
5. Upload file hasil jika ada.
6. Klik kirim.

Setelah dikirim:
- status berubah menjadi `Ditinjau`
- client akan menerima hasil untuk direview

### E. Pantau Pendapatan

Pendapatan pekerja dapat dilihat di:
- `Dashboard`
- `Transaksi`

Jika client menerima hasil:
- sistem akan memproses pendapatan sesuai transaksi yang berlaku

### F. Tarik Saldo

Langkah:
1. Buka menu `Dompet`.
2. Pilih tab `Tarik Saldo`.
3. Isi jumlah penarikan.
4. Isi detail bank atau e-wallet.
5. Kirim permintaan.

Jika penarikan ditolak admin:
- status transaksi akan berubah menjadi `Ditolak`
- alasan penolakan akan tampil di riwayat transaksi

## 4. Panduan Untuk Notifikasi dan Status

Beberapa notifikasi penting:
- `Permintaan top up dikirim`: top up berhasil diajukan dan menunggu admin
- `Permintaan penarikan dikirim`: permintaan withdraw berhasil diajukan
- `Saldo belum cukup`: Anda perlu top up sebelum melanjutkan
- `Transaksi ditolak`: buka menu `Transaksi` untuk melihat alasan penolakan

## 5. Tips Penggunaan

### Untuk Client
- Tulis deskripsi tugas sejelas mungkin.
- Gunakan deadline yang realistis.
- Pastikan budget sesuai tingkat kesulitan tugas.
- Review hasil sebelum menyetujui.

### Untuk Joki / Pekerja
- Ambil tugas yang sesuai kemampuan Anda.
- Kirim catatan hasil yang jelas.
- Upload file hasil dengan nama yang rapi.
- Cek detail tugas sebelum mengirim hasil.

## 6. Jika Mengalami Kendala

Jika mengalami masalah saat menggunakan aplikasi:
- cek kembali koneksi internet
- pastikan data yang diinput sudah benar
- lihat status transaksi atau tugas pada halaman terkait
- hubungi admin platform jika masalah tidak terselesaikan
